# Misc_steg_tool
鉴于CTF内的misc的隐写操作太多太杂，遂有一个整合操作的想法，或许大概可以为解决MISC的部分隐写问题提供可行方案（

程序具有可扩展性，具体可以参照SDK内的md文件

## MainWindow

类似于stegsolve，提供原图、反色、Graybits等变化的查看

# 插件

## Jsteg

提供jpg隐写，采用DCT

## LSB

png的lsb隐写（似乎有点问题）

## SSTV

提供png转sstv和sstv转png

## video_steganography

提供视频隐写，将文件丢入视频中

## zip_security

提供Zip伪加密、解密

Zip隐写
